# GoNav

[English](README.md) | [繁體中文](README.zh-TW.md) | [简体中文](README.zh-CN.md)

GoNav 是一個全端的書籤管理系統，後端使用 Go 語言開發，前端使用 Vue.js。

## 功能特點

- 現代化的書籤管理
- 響應式的UI設計
- 多語系支援
- 明亮/深色模式
- 支援公開書籤分享
- 存取碼限制公開分享

## 專案結構

- `frontend/`: Vue.js 前端應用
- `backend/`: Go 後端應用
